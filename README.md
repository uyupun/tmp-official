# tmp-official

- [uyupun/official](https://github.com/uyupun/official) 開発難航に伴う一時的なオフィシャルサイト

## Setup

- [mise](https://mise.jdx.dev/) を使用して環境を構築する
  - `.node-version` ファイルはCloudflareがNode.jsのバージョンを把握するために残している

```bash
$ mise trust
$ mise install
$ npm install
$ npm run dev
```
